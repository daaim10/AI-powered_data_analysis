# AI-powered_data_analysis
AI-powered data analysis on csv files

## Features

- Upload CSV files directly in the Colab notebook.
- Get **summary statistics**: mean, median, std, missing values, etc.
- Ask **questions about your dataset**, and the AI provides answers using advanced analysis.
- Train and evaluate a classifier on a dataset using the specified target column.
- Train and evaluate a regression model on a dataset using the specified target column.
